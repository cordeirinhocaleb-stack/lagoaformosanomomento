# Execute e implante seu aplicativo AI Studio

Isso contém tudo o que você precisa para rodar seu app localmente.

Visualize seu app no AI Studio: https://ai.studio/apps/drive/1sLwvQ-sXFibQufgSrunLcPyBz4JBK3OB

## Execução Local

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua chave de API do Gemini
3. Inicie o app:
   `npm run dev`
